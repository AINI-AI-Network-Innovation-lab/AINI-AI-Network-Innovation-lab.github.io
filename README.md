# AINI Lab Website

Academic-first lab website for `AINI-AI-Network-Innovation-lab`, built with Astro and static deployment in mind.

## Stack

- Astro static site
- Markdown content collections for `research`, `projects`, and `news`
- JSON data files for `people`, `publications`, and site-wide settings
- GitHub Pages compatible output

## Local Development

```bash
npm install
npm run dev
```

Open the local URL printed by Astro to preview the website.

## Build and Verify

```bash
npm run test
npm run check
npm run build
```
